**WP-Property: Power Tools** adds additional customization and control options to your WP-Property plugin. Use it to Further customize your user interface and control the level of access each user type has to specific content.

### White Label

White Label is a feature that adds even more customization to WP Property. It is located under Property Settings on the Display Tab. This will allow you to rename WP Property components to whatever you want. For example, when we enter Vehicle and Vehicles respectively like so.

### Capabilities Management

After installing Power Tools Add-on you will have a new tab on the WP-Property Settings page. With the Capabilities Tab you are able to easily control the access each type of user has to create, view, edit or delete. For example you can allow agents to edit any property, or keep them restricted to only the ones assigned to them.
